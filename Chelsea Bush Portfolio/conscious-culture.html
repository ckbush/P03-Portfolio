Conscious Culture (2018)

An online shop for sustainably sourced clothing.

Overview

Role: Sketching, Wireframing, UX/UI, Front-end Development

Created for a third year website design and development course, this project is a fully responsive, multi-page website. The concept of the website is an online shop for a sustainable clothing company called Conscious Culture, whose vision is to create simple, modern, and unique clothing items for both men and women. Users are able to purchase products, write reviews, and locate information about the company.

Brand Identity

The first step of this project was to create the company branding components. This included coming up the company's vision, goals, and expectations. Next, the brand idenity was created by producing a logo, as well as fleshing out a unique visual style for elements such as fonts, colours, and imagery that would be used thoughout the website.

Rapid Prototyping

The next step was to create possible user flows on the website. I first sketched to generate quick ideas, and then went in to create medium fidelity wireframes in illustrator. The last part of the prototyping stage was bringing the wireframes into Adobe XD where I added interaction to demonstrate the user flow. Multiple device sizes were considered as each stage, ranging from smart watch to TV console.

Building The Pages

The major component of the project was building out the responsive website. I began by creating a styleguide for website which included the styling of interactive elements, colour palette, grid structure, and fonts. Moving forward I built out simple page templates using HTML and CSS, using placeholders for content before adding in real images and text. Javascript was the final step, which was used to create collapsable dropdown and hamburger menus.

Device Testing & Auditing

The last stage of this project was testing the website on multiple device sizes to ensure the responsiveness was smooth. After debugging any issues from the device testing, I conducted a content audit as well. Through the content audit I recieved useful feeback from my peers where I was able to fix small issues such a font boldness to increase legibility and hierarchy.
