The Project

Created for a third year website design and development course, this project is a fully responsive, multi-page website. The concept of the website is an online shop for a sustainable clothing company called Conscious Culture. Users are able to purchase products, write reviews, and find information about the company.

Rapid Prototyping

The next step was to create possible user flows on the website. I first sketched to generate quick ideas, and then went in to create medium fidelity wireframes in illustrator. The last part of the prototyping stage was bringing the wireframes into Adobe XD where I added interaction to demonstrate the user flow. Multiple device sizes were considered as each stage, ranging from smart watch to TV console.

User Testing & Final Stages

The major component of the project was building out the website using HTML, CSS, and JavaScript. I began by building out a style guide for website which included the styling of interactive elements, colour palette, grid structure, and fonts. Moving forward I built out simple page templates using HTML and CSS and used placeholders for content before adding in real images and text. Javascript was the final step, which was used to create collapsable dropdown and hamburger menus.
